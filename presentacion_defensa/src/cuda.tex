\frame{
	\frametitle{Nvidia CUDA}
	Las GPU Nvidia son arreglos de multiprocesadores, cada uno de los cuales tiene
	\begin{itemize}
		\item varios cores, que ejecutan el mismo programa concurrentemente,
		\item memoria compartida, y mecanismo de sincronización.
	\end{itemize}
	Actualmente NVIDIA ofrece un kit de herramientas de CUDA, las cuales incluyen un compilador, bibliotecas de matemáticas, herramientas para corregir y 
	optimizar redimiento de aplicaciones ejecutadas en la GPU. CUDA es un modelo de programación basado en lenguaje C.
}
